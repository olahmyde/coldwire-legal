# ColdWire — legal

Public Jekyll site for ColdWire store/legal pages. Hosted on GitHub Pages.

- **Live:** https://olahmyde.github.io/coldwire-legal/
- **Stack:** Jekyll + GitHub Pages (custom terminal-noir layout matching the game brand)
- **Contact:** support@icwebpro.net

## Local preview

```bash
cd coldwire-legal
bundle install
bundle exec jekyll serve
# → http://127.0.0.1:4000/coldwire-legal/
```

## Edit the privacy policy

Update `index.md` (source of truth for the live page). `privacy-policy.md` is a plain-text mirror for the main repo's store kit.
